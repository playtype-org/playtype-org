Git is a source control system.

## Quickstart

First download Git for your system

```
apt-get install git
```

### Cloning a repo

First, create and move to an empty directory

```
mkdir code
cd code
```

Next, clone a repo

```
https://github.com/gohugoio/hugo.git
```
